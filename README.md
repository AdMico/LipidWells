# LipidWells v1.0.0 (Updated 01APR26 APM)

**Written by:** Adam Micolich

**Purpose:** Hardware aspects related to our work on 'Organic Electrochemical Transistor Arrays with Integrated Lipid-Sealed Femtolitre Chambers for Simultaneous Electrical and Optical Detection of Membrane Protein Activity'

## Structure

The top level has three folders -- *CNC Parts*, *PCBs & Schematics* and *Photomasks*.

The *CNC Parts* folder has two sub-folders -- *MUXBox*, which contains all the panels for the multiplexer box, and *Stage Parts*, which has all the various components for mounting a coverslip onto a microscope and making the electrical connections to that coverslip.

The *PCBs & Schematics* folder contains five sub-folders which have the PCB designs, Schematics (as appropriate) and the Gerber file sets for production for the six different PCBs used in this project. It also contains a parts list for all the additional components needed for completing the multiplexer box and associated PCBs.

The *Photomasks* folder contains a PDF file with some extended notes about the respective photomask designs.

The photomask files are hosted as a release due to the large file size involved. The photomask release contains 5 compressed RAR files, which contain the GDSII files for production of the photomasks used in this project, with a download size of 238MB (n.b., some of the files, uncompressed, are nearly 1GB in size due to the huge number of components in the design).

Each of the three folders above has its own `README.md` file with further details regarding the contents of those folders.
